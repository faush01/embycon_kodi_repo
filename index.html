Installing Kodi Add-ons

Installing Kodi addons requires two steps, first install the add-on repository and then use the repository to install the desired add-on.

Step One

Install the EmbyCon add-on repository using one of the following options.

Option One - Direct download and install

With this approach you download the repo zip and install it manually.

  1. Download the repository zip file from the link at the top
  2. Save the downloaded repo zip file somewhere Kodi can access it
  3. In Kodi navigate to Add-ons and select "Install from zip file"
  4. If prompted with a security warning about unknown add-ons you will need to turn on "Unknown sources" in the add-on setting
  5. Using the file selector select the repo zip you downloaded
Option Two - Side load from HTTPS

Setup the HTTPS source

  1. In Kodi navigate to the File Manager and select "Add source"
  2. On the "Add file source" dialog select the "Browse" button then select "Add network location..."
  3. On the "Add network location" dialog enter the following:
Protocol: Web server directory (HTTPS)
Server address: faush01.github.io
Remote path: embycon_kodi_repo
  1. Leave everything else and hit the "Ok" button, then hit Cancel and close the dialogs and go back to the home screen
  2. In Kodi navigate to Add-ons and select "Install from zip file"
  3. If prompted with a security warning about unknown add-ons you will need to turn on "Unknown sources" in the add-on setting
  4. You should now see https://faush01.github.io:443/embycon_kodi_repo as one of the sources available, select it
  5. This will show you the repo addon, select it to install
Kodi add network location setup


Step Two

Use the EmbyCon add-on repository to install the desired add-on.

  1. Launch Kodi, open the settings and select Add-ons
  2. Scroll down and select "Install from repository"
  3. Scroll down and select "EmbyCon Addons"
  4. Now select the add-on you want to install
